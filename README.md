# vue-cli-demo
# 汽配供应链顾客端VUE前端

> 该项目仅为前台界面

技术栈：
- vue-cli
- router
- axios
- vuex 
- elementUI

亮点功能（我觉得）：
- dialog多级联动选项框
- 购物车单选、店铺选、全选
- 内容无限滚动
- 支付宝电脑支付接口
- 头像指定目录保存
